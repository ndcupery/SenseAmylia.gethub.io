I'm developing a modular stage design system built around 3D-printed tiles that are rotationally symmetrical, allowing them to be pre-mapped in projection-mapping software before they ever reach a venue. The tiles snap together in flexible configurations on-site, so a cohesive projection-mapped stage can go from cases to show-ready in a fraction of the usual setup time. Beyond the out-of-the-box designs, I can also create custom tiles for clients — incorporating logos, branding, or integrating the system into pre-existing structures and stages.
